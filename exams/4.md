# Exam

1. A developer needs to store session data for a web application running on multiple EC2 instances behind an Application Load Balancer. The session data must be highly available and have low latency.
C. Store sessions in Amazon ElastiCache Redis

2. A Lambda function needs permission to read objects from an S3 bucket.
What is the MOST secure way to provide access?
B. Attach an IAM role to the Lambda function

3. A developer uploads a new version of a static website to an S3 bucket fronted by CloudFront, but users still see old content.
C. Invalidate the CloudFront cache

4. A DynamoDB table is receiving uneven traffic, causing throttling on some partitions.
What is the MOST likely cause?

B. Poor partition key selection\

5. A developer wants an API Gateway endpoint to trigger a Lambda function.
Which integration type is commonly used?

A. Lambda Proxy Integration

6. An application running in Lambda needs temporary credentials to access AWS services.
How are these credentials typically provided?

C. Automatically through the execution role

7. A developer wants to process messages asynchronously between microservices.
Which AWS service is BEST suited?

A. Amazon SQS

8. A Lambda function must process files uploaded to an S3 bucket automatically.
B. Configure S3 Event Notifications to trigger Lambda

9. A developer is using DynamoDB and needs strongly consistent reads.
Which statement is TRUE?

C. Strongly consistent reads cost more throughput than eventually consistent reads


10. A CodePipeline deployment failed during the build stage.
C. CodeBuild

11. An application running on EC2 needs secure access to secrets such as database passwords.
Which solution is BEST?

B. Store passwords in AWS Secrets Manager

12. A developer needs to decouple an application architecture and ensure messages are not lost if consumers are temporarily unavailable.
A. Amazon SQS

13. A Lambda function times out while processing large workloads.
What is a possible solution?

A. Increase the Lambda timeout setting

14. A developer wants to deploy infrastructure using code templates that can be version controlled.

Which AWS service is specifically designed for this?

A. AWS CloudFormation

15. A developer needs to store application logs for monitoring and troubleshooting.

16. A Lambda function needs to connect to an RDS database inside a private VPC.
A. Configure the Lambda function to run inside the VPC

17. A developer wants to automate deployments to EC2 instances with rolling updates.
A. AWS CodeDeploy

18. An SQS queue suddenly receives messages faster than consumers can process them.
What is the benefit of SQS in this scenario?

B. It buffers messages until consumers can process them

19. A developer needs to distribute traffic globally with low latency for static content.
A. Amazon CloudFront

20. A developer wants to invoke a Lambda function whenever a DynamoDB table changes.
Which feature should be used?

C. CloudFront Behaviors
